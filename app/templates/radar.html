What is Radar?

The Radar service is a fast JSON API that gives you meta information about any URL that you pass in. This includes basic details like the title and the description, and more advanced information like the Open Graph and Twitter Card attributes. Here's an example result for an article at The Next Web:

Documentation

GET http://radar.runway7.net?url=[[URL]]

For the full repsonse structure, refer to the example above or the README at github.com/runway7/radar. You can also file feature requests and bug reports on the issues page.

What does it cost?

Radar API calls will be priced at USD $1 per 10,000 calls. The API is currently running free while we set up payment processing.

What can I do with it?

  • Show webpage info and thumbnails when users try to share links
  • See your page through the eyes of Facebook, Google and Twitter
  • Test social integration
  • Provide richer bookmarks
  • Website Research and Analysis
