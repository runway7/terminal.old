What is Blip?

The Blip service is a fast JSON API that gives you meta information about the client that you make the request from, including geolocation and browser identification.

Documentation

GET http://blip.runway7.net

That's it, actually. For the full repsonse structure, refer to the README at github.com/runway7/blip. You can also file feature requests and bug reports on the issues page.

You can simply call the API from a user's browser and either act on the data or pass it back to your server. We make no guarantees about the accuracy of the geolocation, but we've found to be useful and quite precise.

What does it cost?

Blip is completely free. We run the service as a convenience to ourselves and to others. It's also open source under the MIT License, so you're free do deploy your own Blip service if you don't want to rely on ours.

What can I do with it?

  • Quick Geolocation
  • Determine Browser capabilities
  • Analytics
  • Location based personalization
